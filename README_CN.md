## Unity_Demo_Drag3DObject

## Unity_Versions：
- 2017.1.0f3 Or UP

## Explain 说明：

- 实现控制3D物体的拖拽功能

## Use 使用：
- 场景位置： Assets -> PicoMobileSDK -> Pvr_Controller -> Scenes -> Pvr_DragController_Demo
- 按压 **OK按键** 或者 **手柄TouchPad按键** 拖拽3d物体，进行移动

## Announcements 注意事项：
- 拖拽速度过快会使物体脱离控制
